# InteractiveCV 📝

A modern interactive CV that presents professional experience, skills, and projects through a responsive and user-friendly interface built with HTML, CSS, and JavaScript.


# Technologies 🛠️

Responsive design for desktop, tablet, and mobile devices

Interactive navigation between sections

Dynamic display of skills, experience, and projects

Downloadable CV/resume

Contact section

Project portfolio with live links and GitHub repositories

Smooth scrolling and animations

Hover effects and interactive UI elements

Social media and professional profile links

Accessible and user-friendly interface

# The Process 👩🏻‍🍳

I started the process with planning the layout and the structure of the website to ensure its clear and user friendly.

I then focused on developing the website using CSS to produce a responsive and aesthetically pleasing user interface across a variety of devices, while utilising semantic HTML to enhance accessibility and maintainability.

JavaScript was utilised to improve user engagement and offer interactivity with dynamic features and seamless navigation.

In order to enhance usability, accessibility, and overall performance before to create, the application was lastly tested and improved.
